campy.graphics.gcolor module¶

File: gcolor.py¶

This file exports a GColor class that allows students to easily deal with colors.

We deal in a few abstractions:

A GColor should be thought of as an opaque data type that can be initialized from any number of possible representations and serialized to many other types of representations.

# TODO(sredmond): Should a GColor be mutable or immutable? # Decision: immutable!

A representation of a color comes in many forms:
human-readable: a string like “blue” or “seagreen” hex: a string like “#83E” or “#F390D2” int: an argb 32-bit value
