campy.gui.ginteractors module¶

Graphical Interactors similar to those in the Java Swing libraries.

Provides a common interface for constructing GInteractors.

GButton: GCheckbox: GSlider: GTextField: GChooser:

class campy.gui.ginteractors.GButton(label)[source]¶

Bases: campy.gui.ginteractors.GInteractor

An onscreen button.

The following program displays a button that, when pressed, generates the message: “Please do not press this button again!” (credit Douglas Adams):

gw = GWindow()
button = GButton("RED")
gw.add_to_region(button, "SOUTH")

while True:
        e = gevents.wait_for_event(ACTION_EVENT | CLICK_EVENT)
        if e.event_type == MOUSE_CLICKED:
                break

print("Please do not press this button again!")

class campy.gui.ginteractors.GChooser[source]¶

Bases: campy.gui.ginteractors.GInteractor

This interactor subclass represents a selectable list. The GChooser constructor creates an empty chooser. Once the chooser has been created, clients can use addItem to add the options. For example, the following code creates a GChooser containing the four strings “Small”, “Medium”, “Large”, and “X-Large”:

sizeChooser = ginteractors.GChooser()
sizeChooser.addItem("Small");
sizeChooser.addItem("Medium");
sizeChooser.addItem("Large");
sizeChooser.addItem("X-Large");

class campy.gui.ginteractors.GInteractor[source]¶

Bases: campy.graphics.gobjects.GObject

This class is the superclass for all graphical interactors. In most applications, interactors will be added to a control strip along one of the sides of the GWindow, but they can also be placed in specific positions just like any other GObject.
